## 4.3 条件语句

### 4.3.1 条件语句语法格式

```pybnf
if_stmt ::=  "if" assignment_expression ":" suite
             ("elif" assignment_expression ":" suite)*
             ["else" ":" suite]
```

+ 注意使用 `:=` 

```python
if s:= 2+3
	print(s)
```

### 4.3.2 简写格式

```
x = 1
a = 'python' if x>2 else 'physics'
```
